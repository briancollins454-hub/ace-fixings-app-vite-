# Ace Fixings App Prototype (Vite)

## Run locally
1) Install Node.js (LTS)
2) In this folder:
   - npm install
   - npm run dev

## Deploy to Netlify
- Drag/drop this whole folder into a Git repo and connect Netlify
  OR
- Run `npm run build` and deploy with Netlify (Git is easiest).

## Required Netlify Environment Variables
- SHOP_DOMAIN = acefixings.com
- SHOPIFY_API_VERSION = 2025-07
